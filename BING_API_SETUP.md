# Bing Image Search API 申请指南（完全免费）

## 申请步骤（约3分钟）

### 1. 访问 Azure 门户
打开浏览器，访问：https://portal.azure.com

### 2. 登录/注册
- 有微软账号直接登录
- 没有的话免费注册（不需要信用卡）

### 3. 创建 Bing Search 资源
1. 点击左上角的 "创建资源"（+ Create a resource）
2. 在搜索框输入："Bing Search v7"
3. 点击 "Bing Search v7" 卡片
4. 点击 "创建"（Create）

### 4. 填写配置（关键！）
- 订阅: 选择 "免费试用"
- 资源组: 点击"新建" → 输入 "snack-images"
- 名称: snack-image-search  
- 定价层: 选择 "F1 (Free)" ⭐ 免费的！

### 5. 获取 API Key
1. 部署完成后，点击 "转到资源"
2. 左侧点击 "密钥和终结点"
3. 复制 "密钥1" 的值

### 6. 免费额度
- 每月 1000 次查询（我们只需要95次）
- 不需要信用卡
- 永不过期
