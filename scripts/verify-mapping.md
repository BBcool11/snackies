## 如何修正

编辑 `src/utils/imageMapping.json`，添加正确的映射：

```json
{
  "mapping": {
    "001": "091",
    "002": "001",
    "010": "032"
  }
}
```

**说明**：
- `001`（图片编号）→ `091`（数据ID 洽洽瓜子）
- 这样图片 snack-001.png 就会正确显示在 091 号零食上
